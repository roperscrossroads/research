# comparisons

Side-by-side reports comparing 2–4 projects, one file per comparison:

```
comparisons/<topic-or-projects>.md
```

Name by the topic (`self-hosted-file-sync.md`) or the contenders
(`nextcloud-vs-seafile.md`). Start from
[`../templates/comparison.md`](../templates/comparison.md). Every comparison
carries YAML frontmatter and is indexed into [`../INDEX.md`](../INDEX.md).

Per-project reports live in [`../projects/`](../projects/); their machine-readable
records live in [`../catalog/`](../catalog/).
