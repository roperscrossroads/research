# gastownhall/gastown

Steve Yegge's Mad-Max-themed "town" for running 20–30 coding agents: a Mayor
coordinator, per-project Rigs, persistent-identity worker agents (Polecats) with
git-worktree Hooks, a Bors-style merge-queue Refinery, watchdog tiers, escalation,
and DoltHub-federated work ("Wasteland"). Work state lives in a beads ledger. MIT,
~16.4k★.
