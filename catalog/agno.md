# agno-agi/agno

An SDK + runtime ("AgentOS") for agent *platforms*, not just agents: serve agents
as a production API with your own storage, OTel tracing, cron scheduling,
JWT/RBAC multi-tenancy, human-approval gates, and chat/AG-UI/A2A interfaces.
Self-hostable, Apache-2.0, ~41k★.
