# generalaction/emdash

Cross-platform desktop workbench that drives the coding-agent CLIs you already use
(Claude Code, Codex, Gemini, Amp, Copilot…), giving each task its own Git worktree
and branch. Pulls tickets from Linear/GitHub/Jira/GitLab/**Forgejo**/etc.
Local-first (SQLite), YC W26. A human-driver workstation tool, not server-side
automation.
